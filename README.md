# VTuber Analytics v1.0.1

**Analyze. Understand. Improve.**

**Created & Designed by Aino Maria**

VTuber Analyticsは、YouTubeライブチャットをローカル環境で分析し、リスナーの定着状況、配信傾向、コミュニティの状態、要確認コメントなどを振り返るためのWindows向けWebアプリです。

取得したデータや分析結果は、原則として利用者のパソコン内に保存されます。

## 主な機能

- リスナー検索
- 配信参加率、コメント数、ファンスコアの確認
- リスナーのライフサイクル分析
- 新規、復帰、継続中、休眠候補の把握
- 配信カルテ
- 前回配信、同一企画平均との比較
- 企画別の参加人数、コメント数分析
- コミュニティ健康度と定着率
- 月別アクティブリスナー、新規リスナー推移
- 同じ配信に来るリスナーの組み合わせ
- モデレーション候補の抽出
- コメント前後の文脈確認
- 手動判定と対応メモ
- SQLiteによる差分保存
- CSVレポート出力
- データのバックアップ
- ローカルWebアプリ

## 動作環境

- Windows 10 / 11
- Python 3.11以降
- Google Chrome または Microsoft Edge
- インターネット接続
  - チャット取得時のみ必要です
