VTuber Analytics 愛野まりあ版 v0.3 Webアプリ

追加機能
・ローカルWebアプリ
・ダッシュボード
・常連ランキング
・リスナー検索
・リスナーカルテ表示
・企画別分析
・曜日別分析
・最近の配信一覧

特徴
・Flaskなどの追加インストール不要
・Python標準機能だけで動作
・インターネット公開はしない
・自分のPC内だけで利用

導入方法
1. このZIPを解凍します。
2. 中身を VTuberAnalytics_AinoMaria_v0.1 フォルダへコピーします。
3. 同名ファイルは上書きして構いません。
4. 9_Webアプリ起動.bat をダブルクリックします。
5. ブラウザで http://127.0.0.1:8765 が開きます。

終了方法
・黒い画面を閉じる
または
・黒い画面で Ctrl + C

前提
・data/vtuber_analytics.db が存在すること
・v0.2の更新が済み、streamsテーブルにcategoryとweekdayがあること

注意
・このアプリはローカル専用です。
・外部へ公開する設定にはしていません。
