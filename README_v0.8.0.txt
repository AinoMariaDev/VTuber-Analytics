・コミュニティ健康度

導入:
1. ZIPを解凍
2. src / web / VERSION を既存プロジェクトへ上書き
3. Webアプリを終了して再起動
4. ブラウザで Ctrl + F5
5. 「コミュニティ」タブを開く
