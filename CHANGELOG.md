# Changelog

## [1.0.0] - 2026-07-13

### Added
- Listener search, participation rate, fan score, and lifecycle analysis
- Stream reports with previous-stream and category-average comparisons
- Category and weekday analysis
- Community health, retention, dormancy, and co-attendance analysis
- Moderation candidate detection, context review, manual decisions, and notes
- Theme color settings
- CSV reports
- Backups and diagnostics
- Integrated live chat download and one-click data update
- First-run configuration for other VTubers
- Dynamic owner-channel exclusion

### Fixed
- Added support for the current category configuration format
- Fixed `default_category` appearing as a visible category
- Fixed theme color saving
- Fixed settings API errors
- Improved first-run guidance and error messages

### Security and privacy
- Added privacy-focused `.gitignore`
- Excluded local settings, databases, reports, backups, and chat data
- Added security and contribution documentation
