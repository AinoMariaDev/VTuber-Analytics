VTuber Analytics 愛野まりあ版 v0.4 アップデート

追加機能
・直近30配信の参加人数グラフ
・直近30配信のコメント数グラフ
・配信一覧から詳細を開ける
・配信ごとの参加者TOP10
・配信ごとの新規コメント参加者
・リスナー詳細に参加率とタイプ表示
・ダッシュボードの見やすさ改善

導入方法
1. このZIPを解凍します。
2. 中の src と web フォルダを、現在の
   VTuberAnalytics_AinoMaria_v0.1
   フォルダへ上書きコピーします。
3. 9_Webアプリ起動.bat で起動します。

注意
・外部ライブラリは不要です。
・既存のSQLiteデータベースをそのまま使います。
