VTuber Analytics 愛野まりあ版 v0.2 アップデート

追加機能
・配信タイトルから企画を自動分類
・曜日分析
・企画別分析
・リスナーカルテCSV
・HTMLダッシュボード
・ワンクリック分析更新

導入方法
1. このZIPを解凍します。
2. 中身を v0.1 の VTuberAnalytics_AinoMaria_v0.1 フォルダへ上書きコピーします。
3. 7_v0.2分析更新.bat をダブルクリックします。
4. reports/dashboard.html を開きます。

作成されるもの
・reports/企画別分析.csv
・reports/曜日別分析.csv
・reports/リスナーカルテ.csv
・reports/dashboard.html

企画分類ルール
・友人戦
・段位戦
・歌枠
・雑談
・コラボ
・大会
・ホラーゲーム
・その他

分類は config_categories.json で変更できます。
