# YouTube認証・同期

1. Google CloudでYouTube Data API v3を有効化
2. OAuth同意画面を設定
3. OAuthクライアントIDを「ウェブアプリケーション」で作成
4. 承認済みリダイレクトURIへ `http://localhost:8765/api/youtube/oauth/callback` を登録
5. アプリの設定画面でクライアントID・シークレット・リダイレクトURIを保存
6. Googleアカウントを接続
7. 「ライブ配信を同期」を実行

同期対象は、動画ID・タイトル・説明・公開日時・開始終了時刻・配信時間・サムネイルURL・ライブ状態・最終同期日時です。
既存配信は動画IDで更新し、新しい配信だけ追加します。
ライブチャット本文の自動同期はバックグラウンド処理の工程で追加します。
